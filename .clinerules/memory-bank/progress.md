# Progress

## What works
- User authentication (login, registration, password reset)
- CRUD for Cameras
- CRUD for Stores (Shops), including pagination
- Basic dashboard

## What's left to build
- Enhanced dashboard with relevant metrics and charts.
- More sophisticated camera management features (e.g., streaming integration).
- User role management and authorization beyond basic authentication.

## Current status
- Core CRUD functionalities for both Cameras and Stores are implemented and verified.
- Basic navigation is in place.
- Application is stable and deployable via Docker.

## Known issues
- None critical at this time.

## Evolution of project decisions
- Initially, the project focused on camera management. The addition of store management has broadened the scope to provide a hierarchical structure for surveillance infrastructure. This aligns with the product vision of a centralized management hub.
