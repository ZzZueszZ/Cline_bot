# Project Brief

This project, "SurveillanceHub", is a CakePHP 5 application designed to manage stores and their associated cameras. The core functionality involves:

- User authentication (login, registration, password reset)
- Dashboard to provide an overview (currently basic)
- CRUD (Create, Read, Update, Delete) for Cameras
- CRUD for Stores (Shops)

The primary goal is to provide a robust and user-friendly platform for managing surveillance infrastructure within various store locations.
